== Description ==

This plugin allows you to easily create "Content Holders", and generate shortcodes, which can be populated with any text or html you want and placed throughout your website.  You can place the content in either Wordpress content areas or in your php template files simply by inserting the generated shortcode.

== Installation ==

1. Move the unzipped folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
